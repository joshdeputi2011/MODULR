## üìä Understanding Compatibility Scores

### Score Breakdown

Each outfit gets a compatibility score (0-100%):

**Excellent (80-100%)**
- Perfect color harmony
- Appropriate for occasion
- Matching fit/fabric
- Example: White shirt + Navy trousers + Black shoes (Formal)

**Good (60-79%)**
- Good color combination
- Suitable for occasion
- Minor fit/fabric differences
- Example: Sky blue shirt + Khaki trousers + Brown shoes (Casual)

**Fair (40-59%)**
- Acceptable combination
- Some contrast issues
- Works but not optimal
- Example: Black t-shirt + Blue jeans + White sneakers (College)

**Acceptable (0-39%)**
- Wearable but not ideal
- Color theory concerns
- May need adjustments

### Score Components

1. **Color Score (50%)**: Color harmony analysis
2. **Fit Score (25%)**: Matching/complementary fits
3. **Fabric Score (25%)**: Occasion-appropriate materials

## üêõ Troubleshooting

### "No outfits found for this occasion"

**Solutions**:
1. Check if items are tagged for that occasion
2. Ensure you have all categories (top, bottom, shoes)
3. Add more items to your wardrobe
4. Try a different occasion

### "Failed to generate outfits"

**Solutions**:
1. Check internet connection
2. Ensure you're logged in
3. Refresh the page
4. Try again in a few seconds

### Colors not displaying correctly

**Solutions**:
1. Use hex format: #RRGGBB
2. Use the color picker tool
3. Avoid abbreviated hex (use #FFFFFF not #FFF)

### Can't log in

**Solutions**:
1. Check password (min 6 characters)
2. Verify email format
3. Try "Create Account" if you haven't registered
4. Refresh the page
